Tại Sao Chọn Nexus?

⚡

Multi-Head Latent Attention

KV latent compression giảm bộ nhớ cache tới 50% so với MHA truyền thống. Cơ chế absorb tối ưu memory bandwidth.

🧩

Mixture-of-Experts

64 expert với top-6 routing. Mỗi token chỉ kích hoạt subset expert, tiết kiệm compute đáng kể.

⚡

FP8 Quantization

Kernel Triton tùy chỉnh cho FP8 GEMM với autotune. Tăng throughput lên 1.5-2x trên GPU Ampere+.

🔗

YaRN RoPE

Mở rộng ngữ cảnh từ 4K lên 128K+ tokens. Nội suy tần số thông minh với ramp function.

🖥️

Model Parallelism

Chia mô hình trên nhiều GPU. Hỗ trợ Column/Row Parallel + Expert Distribution.

🇻🇳

Tiếng Việt First

Dataset và từ vựng tiếng Việt ~2000+ samples. Xây dựng cho cộng đồng AI Việt Nam.

🏗️ Kiến Trúc Mô Hình

📝 Input Tokens token IDs → embedding
↓
🔤 ParallelEmbedding vocab_size / world_size
↓
🔄 Multi-Head Latent Attention KV latent compression + absorb
↓
⚙️ Feed-Forward Network SwiGLU MLP (dense layers)
↓
🧩 Mixture-of-Experts 64 experts, top-6 routing
↓
🎯 Output Logits lm_head → vocab_size

🚀 Bắt Đầu Ngay

# 1. Cài đặt
pip install -r requirements.txt

# 2. Huấn luyện model tiếng Việt
python inference/build_train.py

# 3. Chat với Nexus!
python inference/chat.py --model inference/chat_model.pt --interactive
# Hoặc huấn luyện nhanh với dữ liệu ngẫu nhiên
python inference/train.py --device cuda --steps 100 --lr 3e-4

# Chạy model đã train
python inference/run_model.py --model nexus_model.pt --prompt "Hello" --steps 50
