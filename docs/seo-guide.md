# Nexus AI 🧠 — Hướng Dẫn SEO & Lên Google

## 🚀 Cách để Nexus lên Google với từ khóa "nexus ai"

### ⚡ Bước 1: Submit lên Google Search Console (LÀM NGAY!)

Mở link này:
```
https://search.google.com/search-console
```

**Các bước:**
1. Đăng nhập bằng Google account của bạn
2. Chọn **"URL prefix"**
3. Nhập: `https://devtantai-coder.github.io/nexus/`
4. Click **Continue**
5. Copy **meta tag** dạng: `<meta name="google-site-verification" content="...">`
6. Mở file `docs/index.html`, tìm dòng:
   ```
   <meta name="google-site-verification" content="YOUR_GOOGLE_SEARCH_CONSOLE_ID" />
   ```
   Thay `YOUR_GOOGLE_SEARCH_CONSOLE_ID` bằng ID thật
7. Commit và push lại
8. Quay lại Search Console, click **Verify**

### ✅ Bước 2: Submit Sitemap

Sau khi xác minh xong:
1. Vào Google Search Console
2. Chọn **Sitemaps** (menu trái)
3. Nhập: `sitemap.xml`
4. Click **Submit**
